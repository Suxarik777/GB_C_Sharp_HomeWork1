## Задачи:
    Задача 10  
    Напишите программу, которая принимает на вход трёхзначное число и на выходе показывает вторую цифру этого числа.
    *Пример:*  
    456 -> 5
    782 -> 8
    918 -> 1     
    Folder in repo: Example010_second_number

    *Пример:*  
    645 -> 5
    78 -> третьей цифры нет
    32679 -> 6

    *Пример:* 
    6 -> да
    7 -> да
    1 -> нет
